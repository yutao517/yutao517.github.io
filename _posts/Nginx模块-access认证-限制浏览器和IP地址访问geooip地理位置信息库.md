## auth_basic认证

htpasswd文件存放路径--->conf目录
htpasswd命令生成htpasswd文件
```bash
		yum provides htpasswd
		yum install httpd-tools -y
		htpasswd -c htpasswd wyt #在conf目录下生成用户名密码文件

```
配置文件添加location
```bash
         location  /mp3 {
            root   html/wyt;
         auth_basic "wyt";
         auth_basic_user_file htpasswd;
         autoindex on;
         }
```
